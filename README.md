# 🎲 Dicee Game

A simple dice game built with **HTML, CSS, and JavaScript**, running directly in the browser.  
Two players roll the dice — the one with the higher number wins.  
Sound effects are included: dice rolling and victory sound for the winner.  

---

## 🚀 Live Demo
👉 [Play here](https://nataliapolak13.github.io/Dicee-Game/)  

---

##  🎮 How to Play
1. Click the **Roll the dice** button.  
2. Both dice will roll and display random numbers.  
3. The player with the higher number wins!  
   - "Player 1 wins!" 🎉  
   - "Player 2 wins!" 🎉  
   - In case of a draw → "Try again!"

---

## ⚙️ Technologies
- **HTML5** – page structure  
- **CSS3** – styling  
- **JavaScript (ES6)** – game logic and sounds
